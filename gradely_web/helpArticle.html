Add Students to a Class

Last Updated: August 10, 2020 04:37

There are several ways you can add or invite students to join your Class. If you are the first teacher assigned by your school to a class, you'll be prompted to add Students using the built-in student account creation tool, but you can also access the student account creation tool later

How to invite students to class

1. Log into your Gradely teacher account.

2. Navigate to the “members” page and click the “Add Students” button to add students to class.

3. Select the class you want to add the students to. P.s. You can only add students to a class you have been assigned to, or approved to join.

4. Set a default password that your students will use to log in for the first time.

5. Enter the student’s name in the required order (first name last name), The parent’s email and Phone number. Providing the parent’s name and phone number is required to so that we can send a notification to the parent that the child has been added on Gradely, and they can take the required next steps.

Click the “Add another Student” button to add another row for student information

6. When you are done adding all your students. Click the “Add all to Class” button to submit. This creates an account for each student, sends an invitation to the parent of the students, and also adds them to your class immediately. You will receive an email containing the login details of the students you just added to the class. This contains each student’s student code along with the default password you created in step 4.

You can also download an excel file containing this information from the email or the alert dialog displayed immediately after completing the process.

7. Click the “Return to Class” button to see the list of all the students you have just added to you class.
