About the Course

During these coding classes, your students will improve their logical thinking, creative thinking, and overall problem solving skills. At the end of the course, your students would have a portfolio showcasing their very own block programming projects or websites or apps.

Requirements

Access to an internet enabled computer, with a fully installed scratch app for block programming and MIT app Inventor for App development

What you’ll learn

Block Programming

  • Terms used in scratch
  • How to set up a scratch profile
  • How to use scratch blocks (Motion, Looks, Events, Control etc).
  • How to add a sprite and backdrop in scratch.
  • How to hide and show a sprite.
  • How to change the size of a sprite.

Mobile & Web App Development

  • Getting Familiar with the MIT APP Inventor Interface
  • How to use the Pallette-Viewer-Component
  • How to use the Image & Label Component
  • Creating a horizontal Arrangement Layout for an android App
  • How to create a Nest Layout
  • Creating a Text-to-Speech Android App
